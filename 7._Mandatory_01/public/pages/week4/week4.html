Package.json

Package.json contains meta data, dependencies and scripts.


Meta data is data that describes and gives information about other data.


Dependencies is the modules that we like to install.


Scripts is a convenient way to bundle common commands.

CSS

CSS stands for Cascading Style Sheets and is the way we style a webpage.

To use the CSS styling we need to link the CSS file to the html file.

This is a way to link the css to html: link rel="stylesheet" href="stylesheet.css"


CSS can also be writen in the html file or we can use inline styling on a html tag to style it directly
